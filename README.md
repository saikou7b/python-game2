# Python Game 2: Unhinged Number Guessing

A number guessing game that actively dislikes the player.

Unlike a normal guessing game, this one occasionally lies, insults your choices, shifts reality without warning, and generally behaves like an unreliable narrator with access to Python.

#Features

* Randomly chosen secret number
* Sarcastic and insulting responses
* "Warmer" and "Colder" hints that may or may not be truthful
* Occasional fake clues
* Dynamic range updates
* Guess history tracking
* Pressure mechanics as attempts run out
* A secret number that sometimes decides to move because consistency is overrated

## How to Play

1. Run the program.
2. Guess a number between 1 and 10.
3. Try to determine whether the game is helping you or gaslighting you.
4. Win before the game runs out of patience.

## Disclaimer

The game contains:

* Mild psychological warfare
* Unreliable hints
* Casual mockery
* Reality instability

Any emotional damage sustained while playing is considered a feature.
